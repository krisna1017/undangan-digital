# 💌 Digital Invitation Website

Website undangan digital interaktif yang dibuat menggunakan **HTML**, **CSS**, **JavaScript**, dan **Bootstrap**.  
Dirancang dengan tampilan modern, responsif, dan mudah digunakan untuk berbagai acara seperti pernikahan, ulang tahun, wisuda, maupun event spesial lainnya.

---

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

---

## ✨ Features

- ⏳ **Countdown Timer**  
  Menampilkan hitung mundur menuju hari acara secara real-time.

- 📩 **RSVP Form**  
  Tamu dapat mengisi konfirmasi kehadiran langsung melalui website.

- 🎵 **Background Music**  
  Musik otomatis/background music untuk menambah suasana undangan.

- 📍 **Location Map**  
  Integrasi Google Maps untuk memudahkan tamu menemukan lokasi acara.

- 📱 **Responsive Design**  
  Tampilan optimal di desktop maupun perangkat mobile.

- 🎨 **Modern UI Design**  
  Desain elegan dan interaktif dengan animasi halus.
